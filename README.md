# 🎓 AI Placement & Internship Assistant

An AI-powered web application that helps students find the best internships based on their skills.

## 🔍 What does this app do?
- Detects skills and gives a Resume Score
- Matches you with internship roles
- Shows top companies hiring for each role
- Generates interview questions based on your skills
- Gives a 30-day learning roadmap
- Provides direct apply links (Internshala, LinkedIn, Forage)

## 🛠 Tech Stack
- Python
- Streamlit
